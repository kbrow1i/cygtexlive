The following packages have been uploaded to the Cygwin distribution:

* texlive-20160520-4
* libkpathsea6-20160520-4
* libkpathsea-devel-20160520-4
* libptexenc1-20160520-4
* libptexenc-devel-20160520-4
* libsynctex1-20160520-4
* libsynctex-devel-20160520-4
* libtexlua52_5-20160520-4
* libtexlua52-devel-20160520-4
* libtexluajit2-20160520-4
* libtexluajit-devel-20160520-4

TeX Live provides a comprehensive, cross-platform TeX system. It 
includes all the major TeX-related programs, macro packages, and fonts 
that are free software, including support for many languages around the 
world.

This release is a rebuild of the TeX Live 2016 binaries and supporting
libraries, built with icu-58.1.
